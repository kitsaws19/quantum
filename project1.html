'Half adder on a Quantum Simulator'


# NOTE:'1+1=2' on a Quantum Simulator
from qiskit import *
from qiskit.tools.visualization import circuit_drawer
from qiskit.tools.visualization import plot_histogram

# Creating a circuit with 3 quantum bits and 2 classical bits
qc = QuantumCircuit(3,2)

# Preparing inputs
qc.x(0) # Comment this line to make Qbit0 = |0>
qc.x(1) # Comment this line to make Qbit1 = |0>
# no changes to Qbit2 (stays |0> always)
qc.barrier()

# Applying AND operation and put result to Qbit2
qc.ccx(0,1,2)
qc.barrier()

# Applying XOR operation and put result to Qbit1
qc.cx(0,1)
qc.barrier()

# Reading outputs
qc.measure(1,0) # Reading XOR value ( sum bit )
qc.measure(2,1) # Reading AND value ( carry-out bit )

counts = execute(qc,Aer.get_backend('qasm_simulator')).result().get_counts()
print(counts)

Steps:

  1. This program shows how we can implement classical circuit on a quantum Computer
  2. Whatever problem you can solve on a Classical Computer can be solved by a Quantum Computer i.e. NP-Completeness
  3. The input is encoded into 'x-gate' and output is measured using 'measure()' on the qubit it is stored.
  4. Here, 1+1=2 in decimals is denoted as '1+1=10' in binary
