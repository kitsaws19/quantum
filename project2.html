#NOTE:This code is for '2+1=3' on a Quantum Simulator.

from qiskit import *
from qiskit.visualization import circuit_drawer
from qiskit.tools.visualization import plot_bloch_multivector
from qiskit.tools.visualization import plot_histogram

# Creating a circuit with 8 quantum bits and 2 classical bits
qc = QuantumCircuit(8,2)

# Preparing inputs
qc.x(0) # Comment this line to make Qbit0 = |0>
qc.x(1) # Comment this line to make Qbit1 = |0>
qc.x(2) # Comment this line to make Qbit2 = |0> ( carry-in bit )
qc.barrier()

# AND gate1 implementation
qc.ccx(0,1,3)
qc.barrier()

# OR gate1 implementation
qc.cx(0,4) 
qc.cx(1,4)
qc.barrier()


# OR gate2 implementation
qc.cx(2,5) 
qc.cx(4,5)
qc.barrier()

# AND gate2 implementation
qc.ccx(2,4,6)
qc.barrier()

# OR gate implementation
qc.x(3)
qc.x(6)
qc.ccx(3,6,7)
qc.x(7)
qc.barrier()

# Measuring and put result to classical bit
qc.measure(5,0) # ( sum )
qc.measure(7,1) # ( carry-out )


counts = execute(qc,Aer.get_backend('qasm_simulator')).result().get_counts()
print(counts)


circuit_drawer(qc, output='mpl')

Steps:

  1. This program shows how we can implement classical circuit on a quantum Computer
  2. Whatever problem you can solve on a Classical Computer can be solved by a Quantum Computer i.e. NP-Completeness
  3. The input is encoded into 'x-gate' and output is measured using 'measure()' on the qubit it is stored.
  4. Here, 2+1=3 in decimals is denoted as '10+1=11' in binary
  5. x-gate is similar to 'NOT gate' in boolean logic, initially all qubits are set to zero.
  6. The CNOT(controlled NOT) is same as 'XOR' gate.
  7. The CCNOT(controlled-controlled NOT) is same as 'AND' gate also called Toffoli gate in qubit gates.
  8. **There is a special gate in Quantum i.e. Hadamard gate(H) which is used to create 'Superposition' of 50% probability of 0's and 1's and much more.
